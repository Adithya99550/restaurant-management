# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

This directory contains Claude Code skills in the `.agents/skills/` folder. Each skill is a specialized capability that can be invoked using the `/<skill-name>` command.

## Available Skills

- **algorithmic-art** - Generate algorithmic visual art and patterns
- **frontend-design** - Frontend design and UI implementation assistance
- **skill-creator** - Create and evaluate new Claude Code skills
- **ui-ux-pro-max** - Advanced UI/UX design and prototyping
- **web-artifacts-builder** - Build and bundle web artifacts

## Usage

Invoke any skill using: `/<skill-name>` (e.g., `/frontend-design`)

Each skill has its own documentation in its SKILL.md file within the skill directory.
